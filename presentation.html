3 / 16
Coherent Mesh Network (CMN)

Architecture

  • 2D mesh topology (scalable 2x2 to 12x12)
  • Cross-Points (XP) form routing fabric
  • CHI protocol for coherency
  • System Level Cache (SLC) at Home Nodes
  • Dynamic clock gating for power management

Node Types

  • RN-F: CPU cores (full coherent)
  • RN-I: GPU, DMA (I/O coherent)
  • HN-F: Home Node (coherency point)
  • SN-F: Memory controllers
  • XP: Cross-point routers
